# Measure Theory: Theoretical Background

## Overview

Measure theory provides a rigorous foundation for integration, probability, and analysis.
The central objects are sigma algebras (collections of measurable sets), measures (assignments
of size/volume), and integrals (accumulated quantities).

## Key Concepts

### Sigma Algebras
A sigma algebra Σ on a set X is a collection of subsets closed under complement and
countable union, containing the empty set. Examples:
- Trivial: {∅, X}
- Discrete: all subsets
- Borel: generated by open sets in a topological space

### Measures
A measure μ on a measurable space (X, Σ) is a function μ: Σ → [0,∞] with:
- μ(∅) = 0 (empty set has zero measure)
- Countable additivity: μ(⊔_n A_n) = Σ_n μ(A_n) for disjoint A_n

### Carathéodory Extension
Every outer measure restricted to its Carathéodory measurable sets gives a complete measure.
This is the primary way to construct measures.

### Lebesgue Measure
The unique translation-invariant measure on ℝⁿ with λ([0,1]ⁿ) = 1. Constructed via
Carathéodory extension from the premeasure λ((a,b]) = b-a.

### Theorems
- Hahn decomposition: signed measure space partitions into positive and negative parts
- Jordan decomposition: signed measure = difference of two positive measures
- Radon-Nikodym: ν ≪ μ ⇒ ν has density dν/dμ
- Lebesgue decomposition: μ = μ_ac + μ_sing (w.r.t. reference measure)
- Vitali covering: countable disjoint balls cover almost everything
- Riesz representation: measures = dual of C_c(X)
