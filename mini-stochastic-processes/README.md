# MiniStochasticProcesses

Stochastic processes theory built on MiniObjectKernel, MiniMeasureTheory, and MiniProbabilityTheory.

## Subpackages
- Core -- StochasticProcess, MarkovChain, BrownianMotion, PoissonProcess
- Morphisms -- ProcessMap, RandomTimeChange, process transformations
- Constructions -- Products, quotients, subobjects, universals
- Properties -- Invariants, preservation, classification data
- Theorems -- Strong Markov property, reflection principle, Birkhoff ergodic theorem
- Examples -- Random walks, two-state Markov chains, Brownian/Poisson properties
- Bridges -- ToAlgebra, ToTopology, ToGeometry, ToComputation
